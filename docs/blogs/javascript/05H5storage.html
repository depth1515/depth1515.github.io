    存储

    zgz 11/25/2018 JavaScript

    # cookie

    • 特点：
      1. 本身用于浏览器和server通讯
      2. 被'借用'到本地存储来
      3. 通过document.cookie = '...'来修改 ( 每次赋值是追加,同个key会覆盖 )
      4. 缺点
        1. 存储大小 最大4kb
        2. http请求需要发送到服务端,增加请求数据量
        3. 只能用document.cookie = '...'来修改, 代码简陋

    # html5存储( localStorage 和 sessionStorage )

    • 特点：
      1. HTML5专门为存储来设计的,最大可存5MB
      2. API简单易用 setItem getItem
      3. 不会随着http请求被发送出去
      4. localStorage 数据会永久存储, 除非代码或手动删除
      5. sessionStorage数据只存在于当前会话, 浏览器关闭则清空

    # 项目应用

    前端项目登录持久化存储，保持登录状态。

    • cookie
    • html5存储( localStorage 和 sessionStorage )
